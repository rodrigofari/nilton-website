Design system

Clean & Premium direction. Madeira palette: warm sand base, atlantic deep navy ink, terracotta and gold accents. Fraunces (display) + Inter (body), self-hosted. Every token in tokens.css is documented below.

Color

tokens.css · color

No pure #000 or #FFF. Backgrounds lean warm sand; ink is atlantic deep navy. Accents pull from volcanic-coast earth (terracotta + gold). Semantic state colors are muted, kept inside the palette so they don't feel like a different system bolted on.

Typography

tokens.css · type

Display: Fraunces (variable, optical-sized — handles editorial italic moments). Body: Inter (Cyrillic-complete; UK locale must not fall back mid-page). Scale: 1.250 (major third). Display sizes lean large with tracking-tight; body keeps comfortable line-height around 1.6.

Font payload status: woff2 files for Fraunces (variable + variable-italic) and Inter (variable + variable-italic) need to be placed in /assets/fonts/ before deploy. See assets/fonts/README.md for the file inventory and Google Fonts download path. Until then, the styleguide renders with system serif/sans fallback — token structure is correct either way.

Motion

tokens.css · duration / easing

Restrained. 200–500ms with the standard easing family. No bouncy, no spring, no overshoot. Hover any card to see the dot translate.
