Nota: Este é um modelo informativo não vinculativo, fornecido de boa-fé. Não foi revisto por um advogado. Se precisares de aconselhamento jurídico vinculativo sobre como este site trata os teus dados, contacta um profissional qualificado em proteção de dados.

Quem gere este site

Este site é operado por Madeira Surf Progress, uma marca de Nilton Freitas — escola de surf licenciada na Madeira, Portugal. Podes contactá-lo via WhatsApp em +351 966 236 416.

O que recolhemos

Recolhemos o mínimo possível. Especificamente:

  • Dados de utilização anónimos — apenas se aceitares cookies de análise. Usamos Google Analytics 4 com anonimização de IP ativada. Isto diz-nos coisas como quantos visitantes o site recebe e que páginas leem. Não te identifica.
  • A tua preferência de idioma — se escolheres uma língua, lembramo-la no localStorage do teu browser para não teres de escolher de novo. Estes dados nunca saem do teu dispositivo.
  • A tua escolha de consentimento — quando aceitas ou recusas cookies de análise, guardamos essa escolha no localStorage do teu browser para não voltarmos a perguntar.

Não recolhemos o teu nome, e-mail, número de telefone ou qualquer outro dado pessoal neste website. Para reservar uma sessão contactas o Nilton diretamente via WhatsApp; essa conversa acontece na plataforma do WhatsApp ao abrigo da política de privacidade deles, não da nossa.

Cookies e armazenamento semelhante

Antes de fazeres uma escolha no banner de consentimento, nenhum cookie é definido e nenhum script de análise corre. Depois de aceitares:

  • Cookies do Google Analytics (_ga, _ga_*) — usados para distinguir visitantes únicos. Retidos até 2 anos conforme a política da Google.

Se recusares, nada do acima corre. A própria escolha de recusa é guardada no localStorage para o banner deixar de aparecer — esse é o único armazenamento nesse caso.

Se o teu browser enviar um sinal Global Privacy Control, tratamo-lo automaticamente como recusa e nunca carregamos análise.

Quanto tempo mantemos os dados

Os dados de análise anónimos são retidos até 14 meses no Google Analytics, sendo depois apagados automaticamente. As entradas em localStorage no teu dispositivo persistem até limpares os dados do browser ou alterares a tua escolha.

Para onde vão os dados

Os dados de análise anónimos são processados pela Google LLC. A Google pode transferir estes dados para fora da UE/EEE. A Google declara ter assinado Cláusulas Contratuais-Tipo para essas transferências, ao abrigo do Artigo 46.º do RGPD.

Nenhum outro terceiro recebe os teus dados a partir deste site.

Os teus direitos ao abrigo do RGPD

Se estás no Espaço Económico Europeu, tens direito a:

  • Aceder aos dados que detemos sobre ti
  • Corrigir dados inexatos
  • Apagar os teus dados
  • Restringir ou opor-te ao tratamento
  • Receber os teus dados num formato portátil
  • Retirar o consentimento em qualquer altura (limpa os dados do browser, ou recusa o banner numa visita futura)
  • Apresentar uma queixa à Comissão Nacional de Proteção de Dados (CNPD) ou à autoridade de controlo do teu país

Como não recolhemos dados identificativos, normalmente não temos nada sobre ti para aceder, corrigir ou apagar. Se tiveres um pedido específico, contacta-nos via WhatsApp.

Alterações a esta política

Se mudarmos algo material, atualizamos a data de "Última atualização" no topo. Não há notificação separada — passa por aqui se visitares com frequência.
